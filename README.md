# Pomodoro Timer

## Run locally

### Method 1: Open the HTML file directly

1. Clone this repository.
2. Open the project folder in File Explorer.
3. Double-click `index.html` to open it in your browser.
4. Keep `app.js` in the same root folder as `index.html` so the script can load correctly.
5. Refresh the page in the browser if you make any changes to the files.

### Method 2: Use VS Code Live Server

1. Clone this repository.
2. Open the project folder in VS Code.
3. Install the Live Server extension if it is not already installed.
4. Right-click `index.html` and choose Open with Live Server.
5. The page will open in your browser and reload automatically when you save changes.

There is no install or build step required because this is a plain HTML, CSS, and JavaScript project.
